# Projet de naissance

PWA d'accompagnement des patientes pour construire leur projet de naissance.

## Démarrage

```bash
npm install
npm run dev
```

Puis ouvrir <http://localhost:5173/#/p/TEST1234> (remplacer `TEST1234` par n'importe quel code).

## Documentation

Voir [`CLAUDE.md`](./CLAUDE.md) pour l'architecture, les conventions et les points d'extension.

## Contenu médical

Édité dans [`src/content/themes.js`](./src/content/themes.js). Les textes s'appuient sur les recommandations HAS et les pratiques des maternités francophones. À valider par le médecin avant mise en prod.
